## Image Optimization Tips

1. **Use online tools:**
   - TinyPNG (https://tinypng.com/)
   - Squoosh (https://squoosh.app/)
   - ImageOptim (Mac)

2. **Recommended formats:**
   - Logos: SVG (scalable, small file size)
   - Photos: JPG or WebP
   - Icons: PNG with transparency

3. **Compression:**
   - Aim for 70-80% quality for JPGs
   - Use progressive JPG encoding
   - Consider WebP for better compression

4. **Retina displays:**
   - Export at 2x size, then optimize
   - Use responsive images with srcset (advanced)
